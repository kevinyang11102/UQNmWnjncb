# Python旅游景点分析系统 python490

## 项目概述

本项目是一款基于Python的旅游景点分析系统，集成了登录注册、旅游数据总览、旅游景点词云分析、旅游景点信息管理以及旅游景点评论可视化分析等多个功能模块，旨在为用户提供一个稳定、高效的旅游景点数据分析平台。

## 技术栈

- 后端：Flask
- 数据库：SQLite
- 前端：HTML, CSS, JavaScript

## 功能模块

### 用户模块

- 登录
- 注册

### 旅游数据模块

- 旅游数据总览

### 分析模块

- 旅游景点词云分析
- 旅游景点评论可视化分析

### 管理模块

- 旅游景点信息管理

## 系统角色

- 普通用户：登录、查看旅游景点数据、发表评论等
- 管理员：管理旅游景点信息、查看用户数据等

## 运行环境

- Python 3.x
- Flask 1.x
- SQLite 3.x

## 部署步骤

1. 安装Python 3.x
2. 安装Flask：`pip install Flask`
3. 安装SQLite 3.x
4. 克隆项目代码到本地
5. 运行数据库迁移：`flask db upgrade`
6. 启动应用：`flask run`
